## Overview

Developed an intuitive and elegant loan EMI calculator that helps users calculate and compare different loan scenarios. The application provides real-time calculations, interactive charts, and the ability to save calculations for future reference.

## Key Features

- **Interactive Calculator**: Built a responsive calculator with real-time updates for principal amount, interest rate, and loan tenure adjustments.
- **Visual Analytics**: Implemented dynamic charts using Chart.js to visualize payment breakdowns and amortization schedules.
- **Loan Comparison**: Created a feature to compare multiple loan scenarios side by side with detailed breakdowns.
- **Save Calculations**: Integrated Firebase authentication to allow users to save and retrieve their loan calculations.
- **PDF Export**: Added functionality to generate and download detailed loan repayment schedules as PDF reports.

## Technologies Used

- **React**: For building the user interface with smooth interactions
- **Firebase**:
  - Hosting: For deployment
  - Firestore: For storing saved calculations
- **Chart.js**: For interactive financial visualizations
- **TailwindCSS**: For creating a responsive and elegant design
- **jsPDF**: For generating downloadable reports

## Challenges and Learnings

The main challenge was creating an intuitive user interface that could handle complex financial calculations while remaining accessible to users with varying levels of financial literacy. This was addressed by implementing clear visual feedback and helpful tooltips. Another challenge was optimizing the performance of real-time calculations and chart updates, which was solved using React's useMemo and useCallback hooks.

## Outcome

The calculator has become a valuable tool for users planning their loans:
- Over 1,000 monthly active users
- Average session duration of 5 minutes
- 40% of users save their calculations for future reference
- Positive user feedback on the intuitive interface and helpful visualizations

This project showcases the ability to create complex financial tools while maintaining an elegant, user-friendly interface that helps users make informed financial decisions.
